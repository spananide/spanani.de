Systembereitstellung

QOLI Installations-Handbuch

1

Python installieren

Python ist die Sprache, in der die KI geschrieben ist. Dein Computer braucht dieses Programm, um den Code lesen zu können.

  1. Lade den Python-Installer über den Link unten herunter.
  2. Öffne die heruntergeladene .exe Datei.
⚠️
Sehr Wichtig! Bevor du im Installer auf "Install Now" klickst, MUSST du ganz unten das Häkchen bei "Add python.exe to PATH" setzen. Ohne dieses Häkchen wird nichts funktionieren!
2

Den Projektordner öffnen

Wir müssen nun die Windows Eingabeaufforderung (CMD) direkt im QOLI-Ordner öffnen.

  1. Öffne den Windows-Ordner, in dem sich die Datei qoli.py befindet.
  2. Klicke ganz oben im Windows Explorer auf die Adressleiste (wo der Dateipfad steht).
  3. Lösche den Text, tippe genau cmd ein und drücke Enter.
3

KI-Bausteine herunterladen

Füge den Befehl in dein schwarzes CMD-Fenster ein (Rechtsklick) und drücke Enter.

C:\Windows\System32\cmd.exe
─
□
✕
C:\QOLI>pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121

Sobald der Download fertig ist, kopiere Befehl 2:

C:\Windows\System32\cmd.exe
─
□
✕
C:\QOLI>pip install ultralytics mss opencv-python
4

Die KI starten

Führe den folgenden Befehl im Terminal aus, um die KI zu wecken.

C:\Windows\System32\cmd.exe
─
□
✕
C:\QOLI>python qoli.py
5

Programm beenden

Um das Programm zu schließen, klicke in das schwarze Terminal und drücke Strg + C.
